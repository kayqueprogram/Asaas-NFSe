# Asaas API Docs - NFSe Guide

Este repositório contém o código-fonte da documentação de integração para emissão de Notas Fiscais de Serviço Eletrônicas (NFSe) via API do Asaas.

A documentação é construída utilizando **VuePress v2**.

## Como executar localmente

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Abra seu navegador no endereço indicado pelo terminal.
